# Airplane Emoji Simulator 💯

Simulate a basic ✈️ airport

## Installation
```
If you need to install yarn: https://yarnpkg.com/lang/en/docs/install/
```
`yarn install`

## Running this bad boy 🛫
`yarn start`

## Testing this baby 🛬
`npm test`
